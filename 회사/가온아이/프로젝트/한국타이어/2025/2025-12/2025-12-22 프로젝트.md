### 한국타이어 전자결재 양식 AS-IS 기반 신규 양식 적용 및 테스트

1. **AS-IS 양식 ID 확인**
    
    - SharePoint 경로에서 작업 대상 양식 ID 확인
        
    - https://o365kaoni.sharepoint.com/.../03.%20ASIS%EA%B2%B0%EC%9E%AC%EC%96%91%EC%8B%9D
        
2. **Arena 기존 양식 HTML 추출**
    
    - Arena 접속 후 FormCode 값만 작업 대상 양식 ID로 변경
        
    - 해당 양식 화면에서  
        `div.dwp-section#subform074_body`  
        → 개발자 도구에서 **outerHTML 복사**
        
3. **로컬 환경 테스트**
    
    - 경로:  
        `ezApproval/Views/DraftUI/ArenaMakeForm.cshtml`
        
    - `<div class="dwp-contents-body">` 내부에 복사한 HTML 전체 삽입
        
    - ezEKP 실행 후  
        `http://localhost:5267/ezApproval/DraftUI/ArenaMakeForm` 접속하여 화면 확인
        
    - 필요 시 아래 항목 수정
        
        - datepicker 처리
            
        - 최상위 div id를 `body`로 변경
            
        - 다국어 처리를 위한 `data-xlang-code` 값 수정
            
4. **폼빌더 반영**
    
    - 관리자 화면 → **폼빌더 사용** 클릭
        
    - **폼빌더 작성기 > HTML 탭** 진입
        
    - 로컬에서 수정 완료된 HTML 전체 붙여넣기
        
    - 폼빌더 함수 정보 작성
        
    - 양식 기본 정보 입력
        
    - 저장 오류 발생 시,  
        기본 `<style>` + `<div id="BodyContent">` 포함된 표준 HTML 구조로 감싸서 재시도
        
    - 저장 완료 확인
        
5. **최종 검증**
    
    - 사용자 화면에서  
        **기안하기 → 결재선 지정 → 상신**
        
    - **미결함 화면**에서 표시되는 양식이  
        AS-IS 양식과 동일하게 노출되는지 최종 확인
